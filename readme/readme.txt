- 스마트폰의 내/외부 저장소에서 txt 파일 읽어오기 기능
- 해당 txt 파일이 카카오톡 대화가 아닐 경우 토스트 메세지 뿌리고 실행 하지 않기 기능
- 사용자들 중 대화 참여자와 자신을 구분하는 기능
- 읽어온 txt 파일을 한 줄 씩 시간, 사용자(대화 참여자), 대화 내용을 구분하여 카카오톡의 UI와 비슷한 형태로 화면에 뿌리기 기능
- 사용자(대화 참여자)의 각 프로필 사진 변경 기능

처음에는 sd카드 폴더에 txt 파일을 넣고 그 파일들 중 하나를 고르는 형식을 구상했으나
API 29부터 getExternalStorageDirectory()가 Deprecated됨.
따라서 sd카드 영역이 아닌 R.raw 폴더에 임의로 파일 저장 후 읽어오는 방식 사용.
